# test.Notes

Homework due to BSC  ios developers position application

## Getting Started

Open `test.Notes.xcworkspace` with Xcode 11.3 or later and run **test.Notes** scheme

### Prerequisites
Xcode with *iOS SDK 13+* is required due to `Combine` framework used

## Running the tests
**[Quick](https://github.com/Quick/Quick)** / **[Nimble](https://github.com/Quick/Nimble)** frameworks used for Unit Testing. Run whole app testing to generate list of tests at `Test Navigator` tab
